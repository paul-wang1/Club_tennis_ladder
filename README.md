# Tennis Ladder Web Application

A web-based tennis ladder system for club tennis teams to manage competitive rankings and match challenges.

## Features

- Separate ladders for men's and women's divisions
- Challenge system (challenge players 2-3 spots above you)
- Automated ranking updates
- Match result reporting
- Player profiles and statistics
- Admin dashboard (coming soon)

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

1. Clone this repository
```bash
git clone <your-repo-url>
cd tennis-ladder-app
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open your browser to `http://localhost:5173`

### Demo Login

Use `john@example.com` to test the app with mock data.

## Deployment

This app is configured to deploy to Vercel:

1. Push your code to GitHub
2. Import your repository in Vercel
3. Deploy!
